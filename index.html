Welcome to ERGpy - Electroretinogram Analysis Software

ERGpy is an automated and open-source software designed for the analysis of electroretinogram (ERG) signals. It leverages the power of SciPy for signal processing and PyQt5 for its graphical user interface (GUI).

Features

  • Automated processing of electroretinogram signals
  • Customizable analysis options:
    • Define start and end regions of the signal to analyze (t0 to t1)
    • Apply Butterworth filters for:
      • Low-pass, band-pass filtering for A, B, and C waves
      • Band-pass and high-pass filtering for oscillatory potentials (OPs)
  • User-friendly graphical interface built with PyQt5
